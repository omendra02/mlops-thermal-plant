🏗️ ISOLATION FOREST ALGORITHM
┌─────────────────────────────────────────────────────────────┐
│                    HOW THE AI WORKS                        │
│                                                             │
│  1. 🎲 Randomly pick a sensor value                        │
│  2. 🎲 Randomly pick a split point                         │
│  3. 🔀 Separate data into two groups                       │
│  4. 🔄 Repeat this process many times                      │
│  5. 📊 Normal data gets separated quickly                  │
│  6. 🚨 Anomalous data takes longer to isolate              │
│  7. 🎯 Use this to detect outliers                         │
│                                                             │
│  Example:                                                  │
│  Normal vibration: 1.1mm/s → Separated in 3 steps         │
│  Anomalous vibration: 8.5mm/s → Separated in 15 steps     │
│                                                             │
│  Result: 8.5mm/s is flagged as anomaly! 🚨                │
